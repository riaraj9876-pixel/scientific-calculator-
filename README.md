# 🎀 Baby Pink Scientific Calculator

## 📌 Description
A GUI-based Scientific Calculator built using Python and Tkinter.  
The calculator performs both basic arithmetic and scientific operations with a custom baby pink themed interface.

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Trigonometric functions (sin, cos, tan)
- Logarithmic functions (log, ln)
- Square root calculation
- Error handling
- Custom baby pink UI design

## 🛠 Technologies Used
- Python
- Tkinter
- Math Module

## ▶ How to Run
1. Install Python on your system.
2. Download the project files.
3. Run the file using:
   python calcu.py
